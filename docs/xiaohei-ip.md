# 小黑 IP 设定文档 — 玩家类型测试专版

> 基于 [Ian Xiaohei Illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) 原始 IP，针对"游戏玩家类型测试"项目进行场景化扩展。
> 核心 DNA 保持不变，新增游戏语境下的动作库、道具库、情绪表达方式。

---

## 一、IP 核心 DNA（继承自原始设定，不可修改）

### 1.1 外观铁律

| 要素 | 规范 | 禁止 |
|------|------|------|
| **身体** | 黑色实心块状，柱形/豆形/盒子形/漏斗形均可，边缘略带手绘抖动 | 不能是完美矢量几何形，不能有渐变填充 |
| **眼睛** | 两个白色圆点，无瞳孔，无高光 | 不能有表情瞳孔、星星眼、泪眼、眯眯眼 |
| **腿** | 两条细黑线，长度约为身体的 0.5-1 倍 | 不能有鞋子、袜子、关节 |
| **手臂** | 偶尔出现，细黑线，只在需要执行动作时画出 | 不能有手指细节、手套、袖子 |
| **表情** | 恒定：空、呆、冷静、认真 | 不能笑、哭、怒、萌、眨眼、张嘴 |
| **身体比例** | 头身比约 1:1 到 1:1.5，整体矮小 | 不能是正常人体比例、不能高瘦 |

### 1.2 人格铁律

- **很认真，但做的事有点荒诞** —— 这是小黑一切行为的总纲领
- 不是吉祥物，不是贴纸，不是可爱装饰
- 是一个低调的系统操作员，有冷幽默
- 有点笨拙但不蠢，像是一个真的在负责某项工作的白板草图小人
- **不卖萌，不扮酷，不装傻** —— 它的幽默来自"认真做荒诞事"的反差

### 1.3 判定标准

> **如果去掉小黑，画面的核心隐喻还能完全成立 → 小黑太装饰了，必须重写 Prompt 让它成为动作主体。**

## 二、游戏语境扩展：玩家小黑

### 2.1 核心设定

在这个项目中，小黑不再只是"系统操作员"，而是一个**玩家身份的具象化**。

- 小黑 = 各类玩家的灵魂投射
- 小黑的黑色实心身体 = 玩家的"游戏人格本体"
- 白点眼 = 玩家面对屏幕时那种专注但空洞的眼神
- 细腿 = 玩家在现实中的存在感（很细，因为主要活在游戏里）

### 2.2 游戏专属视觉词汇

#### 小黑 + 游戏设备的交互方式

| 设备 | 小黑的交互方式 | 注意 |
|------|---------------|------|
| **鼠标/键盘** | 细线手臂末端分叉成两股握住鼠标；键盘被画成简笔矩形，小黑的细腿踩在巨大的键帽上 | 设备要画得比小黑大，营造"被游戏吞噬"的比例感 |
| **手柄** | 手柄画成简笔椭圆+十字键，小黑双手举着（手柄和小黑身体差不多大） | 不能画成真实手柄的精细度 |
| **手机** | 简笔圆角矩形，小黑站在手机上面或趴在屏幕前 | 手机屏幕的白光映在小黑的白点眼上（极简表达） |
| **屏幕/显示器** | 简笔矩形框，里面用极简符号表达游戏内容（血条、分数、像素小人） | 屏幕内容不超过 5 个元素 |

#### 小黑 + 游戏情绪的视觉语法

小黑的**表情永远不变**（白点眼 + 空表情），情绪通过**身体姿态和周围符号**表达：

| 情绪 | 身体语言 | 画面符号 |
|------|---------|---------|
| **焦虑/压力** | 身体微微弯曲/变形，有细密的抖动辅助线 | 红色感叹号漂浮、计时器、下滑的箭头 |
| **专注/沉迷** | 身体前倾，白点眼离屏幕极近 | 周围世界褪色（用虚线表示被忽略的现实） |
| **快乐/满足** | 身体膨胀一点点（还是黑色实心，但比例微调至 1:1.2） | 极小的星星或音符（不超过 3 个，克制使用） |
| **固执/偏执** | 身体变成一个顽固的方块，腿缩得很短 | 重复的行为轨迹线（橙色环状箭头） |
| **迷茫/探索** | 身体微微倾斜，头顶飘一个"？" | 周围有岔路或迷雾（虚线表示未知区域） |
| **愤怒/不满** | 身体膨胀到 1:1.3，细腿跺地（画两道震动线） | 红色"？"从头顶冒出，但表情依然空白 |

> **关键原则**：情绪用身体语言和画面符号表达，**永远不改小黑的表情**。这是冷幽默的核心——一个面无表情的小黑在暴怒跺脚，比画一个愤怒表情好笑十倍。

### 2.3 小黑的变形能力

在游戏语境下，小黑可以根据"玩家类型"进行有限的形态变形：

| 变形类型 | 描述 | 适用场景 |
|---------|------|---------|
| **仓鼠球形态** | 身体变成巨大圆球（被塞满），内部隐约可见物品轮廓 | 仓鼠类型 |
| **磨盘驴形态** | 身体拉长成柱形，脖子上挂绳连到磨盘 | 赛博驴类型 |
| **卡墙形态** | 半个身体陷进墙里（锯齿破面线表示穿模） | BUG启动器类型 |
| **保护伞形态** | 身体展开成伞状/树状，罩住更小的迷你小黑 | 游戏奶妈类型 |
| **打坐形态** | 身体变成扁圆形（盘腿），周围漂浮装饰物 | 挂机仙人类型 |
| **展示柜形态** | 身体正面有玻璃陈列窗，里面放着金光闪闪的物品 | 氪佬类型 |

> **变形规则**：始终保持黑色实心填充 + 白点眼 + 细腿。变形是轮廓的改变，不是材质或颜色的改变。

## 三、12 类型的小黑专属姿态

每个玩家类型对应一个标志性的小黑姿态，用于结果页主视觉。

### 3.1 分奴 · 血压仪前的囚徒

```
小黑坐在巨型"分数血压仪"前，一根管子从头连到仪器。
仪表指针在红色危险区抖动。小黑双手紧握鼠标，
身体前倾到极限（但白点眼表情依然空白）。
脚边散落几个被捏碎的写着"娱乐"的球体。
```

**核心隐喻**：被分数绑架的人，娱乐已被碾碎。
**小黑在干什么**：面无表情地盯着血压仪，双手死死攥着鼠标。

### 3.2 压力怪 · 问号风暴中心

```
小黑站在由无数黄色"?"组成的龙卷风中心。
一只手指疯狂敲击一个被按到凹陷的巨大信号按钮。
从小黑头上飘出一个巨大的对话气泡，里面只有一个"?"。
脚下的地面散落着被"?"符号砸扁的更小小黑。
右上角有一个小小的"屏蔽"按钮——小黑选择无视它。
```

**核心隐喻**：暴风眼里的施暴者，也是自己情绪的囚徒。
**小黑在干什么**：在"?"风暴中心面无表情地狂按信号键。

### 3.3 乐子人 · 倒立玩家

```
小黑倒立着用手柄玩游戏，两条细腿在空中闲适地晃。
屏幕上是一场团战，小黑操控的角色在角落里独自跳舞。
屏幕旁四个队友的愤怒头像气泡飘过来，
但小黑头顶只有一个音乐符号。
地上有几面被踩碎的"认真""Meta""胜率"小旗子。
```

**核心隐喻**：世界在认真打仗，他在认真跳舞。一样认真，方向不同。
**小黑在干什么**：倒立打游戏，面无表情地让角色在团战里跳舞。

### 3.4 赛博驴 · 永动磨盘

```
小黑（身体拉成柱形）拉着一台巨大的石磨转圈。
石磨上刻着"每日任务"，磨盘流出来的不是面粉，
而是极微量的"快乐"颗粒（小到几乎看不见）。
小黑脖子上挂着打卡机，记录纸上全是密密麻麻的√。
身后墙上贴着一张精确到分钟的"日程表"。
远处有一扇门写着"卸载"——但小黑被拴在磨盘上。
```

**核心隐喻**：自我驯化的劳动者，把游戏玩成了KPI。
**小黑在干什么**：像驴一样拉磨，面无表情地完成第 847 次日常。

### 3.5 仓鼠 · 爆裂收纳球

```
小黑的身体膨胀成一个巨大的圆球（仓鼠球形态）。
球体透明可见内部塞满了药水瓶、装备、材料、钥匙——
所有物品挤到变形，球壁已经出现裂纹。
但小黑还在用细线手臂从外面拼命往球里塞更多东西。
一个小 Boss 站在球旁边（画得很小，不起眼），
小黑完全无视它，专注在"捡东西"上。
球体上方飘着："背包已满（第 99 次）"
```

**核心隐喻**：囤积的安全感本身就是最大的不安全。
**小黑在干什么**：快要爆炸了但仍然在往自己身体里塞东西，无视面前的 Boss。

### 3.6 白金人 · 奖杯外科医生

```
小黑站在一个巨大奖杯陈列柜前，柜子有无数的格子。
小黑一手拿镊子一手拿放大镜，
像外科医生一样极其精密地把一枚小小的白金奖杯放进一个空格。
地面上散落着游戏光盘盒，全部贴着"已白金"标签。
小黑的细线手臂上挂着各种"刷杯专用装备"（计数器、路线图）。
日历翻开，每一页都写着"今日白金目标：XXX"。
```

**核心隐喻**：对"完成"的执念已经精细到外科手术级别。
**小黑在干什么**：用镊子精密放置最后一枚白金奖杯，面无表情但极度专注。

### 3.7 该溜子 · 反导航体质

```
小黑站在巨大开放世界地图的正中央。
从小黑脚下延伸出无数条弯曲的红色虚线探索轨迹，
遍布地图每个角落——但每一条都精准绕开了
地图中央标注的"主线任务★"。
小黑正举着望远镜望向最远的那座山。
身后站着一个举着"HELP!"旗子的小 NPC（焦急），
但小黑完全背对着它。
小黑的身上粘着草屑、树枝、奇怪的蘑菇——探索的勋章。
```

**核心隐喻**：方向感用在了一切"不该去"的地方。
**小黑在干什么**：举着望远镜看最远的山，背后 NPC 等了 150 小时。

### 3.8 考古学家 · 碑文前的学究

```
小黑头戴一顶极简的探险帽（还是黑色实心，帽檐是两条短线）。
蹲在一块巨大的石碑前，一手放大镜一手疯狂做笔记。
笔记堆成了一座小山（每张纸上密密麻麻的手写文字）。
从小黑身后延伸出一条虚线，连着一个坐在王座上打哈欠的 Boss。
Boss 头顶飘着云："等了 5 小时..."
小黑脚下散落着"剧情不跳"、"先看对话"的小标签。
```

**核心隐喻**：在需要战斗的游戏里选择了学术生涯。
**小黑在干什么**：蹲在石碑前疯狂记笔记，完全忘记有个 Boss 在等它。

### 3.9 BUG 启动器 · 墙体共生体

```
小黑半个身体卡在一面墙里——上半身已经穿模进去，
只剩两条细腿在外面乱蹬。
墙面画着锯齿状的"破面"线条，表示碰撞体缺失。
地面有一块浮空的石头（违反物理法则）。
小黑身体旁边漂浮着一个剪贴板，写着"Bug #47：墙角碰撞体缺失"。
背景里有一扇正常的门（小黑完全可以走门），
但门上贴着一张纸条："太无聊了"。
```

**核心隐喻**：已知的门不走，偏要测试未知的墙。
**小黑在干什么**：半截身体卡在墙里，仍然认真地记录 Bug 编号。

### 3.10 游戏奶妈 · 人形保护伞

```
大黑站在画面中央，展开细线双臂（尽量张开），
像一棵树/一把伞一样罩着脚下一群迷你小黑。
迷你小黑的眼睛是困惑的弯线（¯.¯），身体更小。
大黑手里拿着药瓶、盾牌、地图、新手教程手册。
从大黑身体里延伸出无数橙色细线，连接到每一个迷你小黑。
大黑自己的血条只剩一丝（红色批注："我血条呢？"）。
但表情依然是认真的、空白的——它在做自己觉得该做的事。
```

**核心隐喻**：燃烧自己照亮萌新，血条空了也没发现。
**小黑在干什么**：张开双臂保护一群萌新，自己的药全分出去了。

### 3.11 挂机仙人 · 主城景观石

```
小黑盘腿坐在主城喷泉广场的正中央（最显眼的位置）。
身体纹丝不动，像一块景观石。
数件华丽的外观物品漂浮在身体周围：
翅膀、光环、披风、头饰（全部简笔线条勾勒）。
面前一个计时器显示"在线：7h 55min，战斗：5min"。
头顶一个巨大的对话气泡，里面是密密麻麻的聊天记录。
远处副本入口发着光，通往入口的路标写着"800m → 太远了"。
小黑脚下围了一圈正在欣赏它外观的更小小黑。
```

**核心隐喻**：在线时长全服前10，战斗时长全服倒数第一。
**小黑在干什么**：在主城 C 位打坐，面无表情地聊了 8 小时天。

### 3.12 氪佬 · 博物馆馆长

```
小黑站在一个巨大的宝库/博物馆中央。
四面墙上挂满金光闪闪的物品：传说武器、限定皮肤、绝版道具——
每一件都在展示柜里完好无损（标签写着"已拥有"）。
小黑正拿着一个巨大的刷卡机（POS机简笔画），
面无表情地刷下又一件展品。
角落里堆着一个积灰的"新手任务"卷轴（还没打开）。
小黑身上挂着一个很小的等级牌：Lv.1。
宝库门外是"实际游戏内容"区域，门上挂着一个牌子："改天"。
```

**核心隐喻**：拥有的快感替代了游玩的快感。
**小黑在干什么**：在私人博物馆里刷卡买第 48 件传说装备，等级仍然是 1。

## 四、游戏专属道具库

小黑在画面中可以交互的游戏相关物品，全部用极简手绘线条表达：

### 4.1 设备类
- **显示器**：简笔矩形 + 细线底座，屏幕内不超过 5 个像素级元素
- **键鼠**：键盘 = 横向矩形 + 几条横线表示键位；鼠标 = 椭圆 + 一条线
- **手柄**：椭圆 + 十字键（两条交叉短线）+ 两个小圆按钮
- **手机**：圆角矩形，屏幕白光

### 4.2 游戏 UI 类（仅在屏幕内出现）
- **血条**：一条红色横线 + 黑色轮廓
- **分数/段位**：一个徽章形状 + 数字/箭头
- **信号标记**：黄色"？"（最常见的视觉元素之一）
- **对话气泡**：椭圆形 + 短文字
- **Mini 地图**：角落的小矩形，里面几个圆点

### 4.3 情绪符号类
- **？**：黄色或红色，手写质感，大小不一
- **！**：红色，用于警告/强调
- **✓**：红色，用于"已完成"标记
- **虚线**：橙色，表示路径/轨迹/探索路线
- **裂纹**：黑色锯齿线，表示破坏/穿模/压力

### 4.4 游戏物品类
- **药水瓶**：简笔三角瓶 + 十字标记
- **装备/武器**：极简轮廓，不能画细节
- **宝箱**：矩形 + 半圆顶 + 锁扣
- **金币/资源**：小圆圈 + 闪光短线
- **奖杯**：U 形杯 + 两个小耳朵

## 五、构图铁律

### 5.1 比例规则
- 小黑占画面 **40-60%**，这是硬指标
- 留白至少 **35%**，推荐 40%+
- 批注文字每处 **2-8 字**，整图 5-8 处
- 画面只表达 **一个核心隐喻**，不堆叠多个概念

### 5.2 色彩规则
| 颜色 | 用途 | 占比限制 |
|------|------|---------|
| **黑色** | 主体、线条、轮廓、小黑身体 | 画面 15-25% |
| **白色** | 背景（必须是纯白 #FFF） | 画面 35-50% |
| **红色** | 重点批注、警告、情绪点、关键提醒 | ≤ 5% |
| **橙色** | 路径、流程、箭头、连接线、自动化 | ≤ 5% |
| **蓝色** | 次要注释、心理状态、第二层解释 | ≤ 3%（可选） |

### 5.3 禁止清单（完整版）

**绝对不要出现在画面里：**
- ❌ 米色/暖灰/纸纹/渐变/阴影背景
- ❌ 商业插画感、3D 渲染感、矢量扁平感
- ❌ PPT 信息图、流程图、思维导图、架构图
- ❌ 可爱卡通、萌系、儿童绘本风
- ❌ 任何标题文字出现在画面左上角
- ❌ 复杂表情（笑脸、哭脸、怒脸、星星眼）
- ❌ 小黑穿衣服、戴复杂饰品、有头发
- ❌ 真实设备/产品/UI 的精细描绘
- ❌ 渐变、高光、投影、发光效果
- ❌ 花哨字体、标题字体、衬线字体
- ❌ 小黑只是站在角落看——它必须是动作主体

## 六、与 12 类型的视觉一致性

### 6.1 类型 → 视觉关键词速查

## 七、Prompt 快速模板

### 单图生成模板

```
16:9 横版中文正文配图。
纯白背景（不是米色不是暖灰，没有任何纸纹、阴影、渐变）。
黑色手绘线稿风格，线条纤细略带抖动，不是机械矢量线条。
画面大量留白，主体占画面 45-55%。

[在此插入该类型的"小黑在干什么"动作描述 + 核心隐喻描述]

小黑（黑色实心块状小怪物，白色圆点眼睛，细黑线腿，
表情永远是空的、呆的、认真的）必须是画面的核心动作执行者。
它的情绪通过身体姿态和周围符号表达，面部永远不改。

[5-8处中文手写批注，分配红/橙/蓝色]

不要商业插画感、不要PPT信息图、不要可爱卡通、
不要复杂架构图、不要标题文字在左上角。
线条要怪诞、有趣、简洁清爽、天马行空。
```

### 使用方式

1. 从第三章选取对应类型的"小黑在干什么"描述
2. 从第六章速查表获取该类型的核心道具和隐喻
3. 拼入模板，生成完整 Prompt
4. 生图后用第六章的禁止清单逐项检查
