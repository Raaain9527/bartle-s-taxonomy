# 12 种玩家类型 — 小黑怪诞插图 Prompt

> 基于 [Ian Xiaohei Illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) 风格体系
> 所有插图统一规格：16:9 横版，纯白背景，黑色手绘线稿，少量红/橙/蓝中文手写批注

---

## Prompt 通用前缀（所有类型共用）

```
16:9 横版中文正文配图。纯白背景（不是米色不是暖灰，没有任何纸纹、阴影、渐变）。
黑色手绘线稿风格，线条纤细略带抖动，不是机械矢量线条。
画面大量留白，主体占画面 40-60%。
小黑（黑色实心小怪物，白色圆点眼睛，细腿，偶尔有细手臂，身体呈柱形/豆形/盒子形）
必须是画面的核心动作执行者，不是装饰。
少量红色、橙色手写中文批注（每处 2-8 字），蓝色仅作次要注释。
不要商业插画感、不要 PPT 信息图、不要可爱卡通、不要复杂架构图、
不要标题文字出现在左上角。
```

## 1. 分奴（Score Slave）

### 画面描述
小黑坐在一台巨大的"分数血压仪"前。一根管子从仪器连到小黑的头上，仪表的指针在红色危险区疯狂抖动。小黑双手紧握鼠标/手柄，身体紧张到变形（柱形身体微微弯曲）。屏幕上显示一个巨大的 Rank 标志和向上/向下的箭头（目前方向：↓）。小黑脚边散落着几个被捏碎的"娱乐"球体。

### 画面构成
- **主体（50%）**：小黑 + 血压仪 + 屏幕
- **红色批注**："再输一把就睡"（旁边小字：第 8 次）、"血压↑"
- **橙色批注**：从屏幕到小黑画一条橙色箭头，标注"胜负反馈"
- **蓝色批注**："T0 only"

### 小黑在干什么
小黑在测量自己的 Rank 血压，同时死死盯着屏幕上的分数变化，身体被压力压弯。

## 2. 压力怪（Pressure Monster）

### 画面描述
小黑站在一个由无数黄色"?"号组成的风暴中心。它的手指（细线状）正在疯狂敲击一个巨大的信号按钮（按钮已经被按到凹陷）。从小黑嘴里延伸出一个对话气泡，气泡里只有一个巨大的"?".它周围的地面上散落着被"?"砸扁的其他小人。画面右上角有一个很小的"屏蔽"按钮，小黑选择看不见它。

### 画面构成
- **主体（50%）**：小黑 + 信号风暴 + 巨大对话气泡
- **红色批注**："信号键已磨损"、"?"（巨大的红色问号漂浮）
- **橙色批注**：画一条橙色路径从小黑 → 队友（被砸扁的小人），标注"问责路径"
- **蓝色批注**："已屏蔽 200+"

### 小黑在干什么
小黑正在进行一场"?"暴风雪式的战术沟通（自己觉得是在沟通）。

## 3. 乐子人（Chaos Gremlin）

### 画面描述
小黑倒立着用手柄玩游戏，脚在空中晃。屏幕上是一场团战，但小黑选的角色正在角落里跳舞（不是挂机，是主动跳舞）。屏幕旁边有四个队友的愤怒小头像气泡，但小黑头上飘着一个巨大的音乐符号和"开心"两字。画面里有几个被踩碎的"胜率""Meta"标签。小黑的脚边有一面倒下的旗子写着"认真"。

### 画面构成
- **主体（50%）**：倒立的小黑 + 屏幕上的跳舞角色
- **红色批注**："快乐就完了"、"胜率 43%"
- **橙色批注**：从小黑到跳舞角色画一条橙色旋风，标注"游戏理解"
- **蓝色批注**："第 7 次科研出装"

### 小黑在干什么
在竞技游戏里倒立着玩，并且让自己的角色在团战时跳庆祝舞蹈。

## 4. 赛博驴（Cyber Donkey）

### 画面描述
小黑拉着一台巨大的石磨在转圈。石磨上写着"每日任务"。从石磨里流出来的不是面粉，而是微量的"快乐"颗粒（少得可怜）。小黑的脖子上挂着一个打卡机，打卡记录纸上密密麻麻全是√。小黑身后是一个巨大的"日程表"，精确到分钟。远处有一扇门写着"卸载"，但小黑被拴在石磨上。小黑的表情依然是平静认真的（它在很严肃地做这件事）。

### 画面构成
- **主体（55%）**：小黑 + 石磨（日常任务磨盘） + 打卡机
- **红色批注**："第 847 天"、"快乐产量：极低"
- **橙色批注**：环绕石磨画橙色循环箭头，标注"无尽的日常"
- **蓝色批注**："活动结束倒计时 3 天"

### 小黑在干什么
小黑像驴一样拉着日常任务的磨盘转圈，同时脖子上挂着打卡机。

## 5. 仓鼠（Hamster）

### 画面描述
小黑的形态变成了一个巨大的仓鼠球——身体圆滚滚（被塞满的），里面塞满了各种药水瓶、装备、材料、钥匙、任务物品。球已经膨胀到快要爆炸，但小黑还在从外面拼命往球里塞更多东西。球的外面站着一个小 Boss（很小，不起眼），小黑完全无视 Boss，专注在"捡东西"这个核心任务上。小黑的白色圆点眼睛透出一种"还不够"的偏执。

### 画面构成
- **主体（55%）**：仓鼠球形态的小黑 + 溢出边缘的物资
- **红色批注**："最后一个"、"万一有用"、"背包已满(第99次)"
- **橙色批注**：画橙色箭头指示小黑从外部 → 球内塞东西的动作循环
- **蓝色批注**："已拥有：999+"

### 小黑在干什么
小黑变成了一个被物资撑到即将爆炸的仓鼠球，但仍然在把外界的所有东西塞进自己身体里。

## 6. 白金人（Platinum Hunter）

### 画面描述
小黑站在一个巨大的奖杯陈列柜前。陈列柜有无数格子，有的亮着（已获得），大部分黑着（未获得）。小黑正拿着镊子和放大镜，极其仔细地往一个格子里放置一枚小小的奖杯。地面上散落着无数游戏光盘盒，上面都贴着"已白金"标签。有一个日历翻开着，每一页都写着"今日目标：白金 XXX"。小黑的身体变得像工具台一样——手臂上挂着各种"刷奖杯专用装备"。

### 画面构成
- **主体（50%）**：小黑 + 奖杯陈列柜 + 镊子放大镜
- **红色批注**："还差一个杯"、"100% or nothing"
- **橙色批注**：画橙色线条连接"已白金的游戏"→ 陈列柜亮格
- **蓝色批注**："PSNProfiles 常驻"

### 小黑在干什么
小黑像外科医生一样用镊子和放大镜，精密地把最后一枚白金奖杯放进陈列柜。

## 7. 该溜子（Digital Wanderer）

### 画面描述
小黑站在一个巨大开放世界地图的正中央。从小黑脚下延伸出无数条弯曲的探索轨迹（红色虚线），像迷宫一样遍布整个地图——但每一条都巧妙地避开了地图上标注的"主线任务"标记。小黑正拿着望远镜望向最远处的山。它的身后有一个焦急的小 NPC 举着"请来救我"的旗子，但小黑完全背对着 NPC。小黑的身上粘满了草、树枝、奇怪的蘑菇（探索的勋章）。

### 画面构成
- **主体（50%）**：小黑 + 望远镜 + 布满探索轨迹的地图
- **红色批注**："那边有什么？"、"主线不急"
- **橙色批注**：用橙色虚线画出小黑所有的探索路径（全部绕开主线）
- **蓝色批注**："游戏时长 200h，主线进度 3%"

### 小黑在干什么
小黑举着望远镜望向地图最远的角落，身后焦急的 NPC 和"主线"标记被完全无视。

## 8. 考古学家（Archaeologist）

### 画面描述
小黑戴着一顶小小的探险帽（还是黑色实心身体，只是头上多了一个简笔画帽子形状），蹲在一块巨大的石碑前。石碑上刻满了密密麻麻的游戏世界文字。小黑一手拿着放大镜仔细研读，另一只手在疯狂做笔记（笔记已经堆成一座小山）。从小黑后面延伸出一条线，连着一个正在苦等主角的 BOSS——BOSS 坐在王座上打哈欠，头上飘着"等了 5 小时"。小黑完全沉浸在石碑的文字中。

### 画面构成
- **主体（50%）**：小黑 + 石碑 + 笔记山
- **红色批注**："这段故事很关键"、"300年前..."
- **橙色批注**：从小黑 → 笔记山画一条橙色连接线，标注"考据输出"
- **蓝色批注**：BOSS 头上冒出："他还在读？"

### 小黑在干什么
小黑在研读一块游戏世界里的石碑，笔记堆成了山，完全忘记了 BOSS 还在等它。

## 9. BUG 启动器（Bug Conjurer）

### 画面描述
小黑正费力地把自己挤进一面明显不该进去的墙缝里。半个身体已经穿模进墙里，剩两条细腿在外面挣扎。画面里的世界出现了视觉裂缝——墙面有锯齿状的"破面"线条，地面有一块浮空了。小黑身体旁边飘着一个剪贴板，上面记录着"第 47 号 Bug：墙角碰撞体缺失"。背景里有一个正常的游戏门（小黑完全可以走门），但门上贴了一个"太无聊了"的纸条。

### 画面构成
- **主体（50%）**：卡在墙里的小黑（半个身体穿模）
- **红色批注**："能卡进去！"、"第 47 号 Bug"
- **橙色批注**：画橙色箭头指向碰撞体裂缝，标注"漏洞位置"
- **蓝色批注**："正常入口 →"（指向那扇被无视的门）

### 小黑在干什么
小黑正在把自己塞进一面不该穿过的墙，并在剪贴板上认真地记录这个 Bug。

## 10. 游戏奶妈（Game Nanny）

### 画面描述
小黑站在画面中央，展开双臂（细手臂尽量张开），像一把保护伞一样罩着脚下几个更小的小黑（萌新版——身体更小，眼睛是困惑的曲线而非圆点）。大黑手里拿着各种辅助道具：药水、盾牌、地图、新手教程手册。从大黑身体里延伸出无数条线，每一条线连接着一个萌新小黑——像一棵树的根系。大黑自己的血条只剩一丝（因为它把药都分给了别人）。表情依然是认真平静的。

### 画面构成
- **主体（50%）**：大黑张开双臂 + 脚下/周围的小小黑
- **红色批注**："不用怕，有我在"、"我血条呢？"
- **橙色批注**：画橙色线条从大黑辐射到每一个萌新小黑，标注"带人网络"
- **蓝色批注**："好友数量：上限"

### 小黑在干什么
大黑张开双臂保护着一群萌新小小黑，把自己的药全分了出去，自己的血条快空了。

## 11. 挂机仙人（AFK Immortal）

### 画面描述
小黑盘腿坐在主城的喷泉广场正中央——一个极其显眼的位置。它的身体纹丝不动，但被数件华丽的衣服、翅膀、光环层层包裹（所有外观物品漂浮在身体周围）。小黑面前有一个计时器显示"在线：7 小时 55 分钟"。计时器旁边有个"战斗"统计：5 分钟。小黑头上飘着一个巨大的对话气泡，气泡里是密密麻麻的聊天记录。远处有一个副本入口在发光，但通往入口的路上写着"太远了不去"。小黑脚下围了一圈正在欣赏它外观的小小黑。

### 画面构成
- **主体（50%）**：盘腿静坐的小黑 + 华丽外观漂浮层 + 聊天气泡
- **红色批注**："在线 8h，战斗 5min"、"新外观 get"
- **橙色批注**：画橙色圈环绕小黑所在的位置，标注"主城 C 位"
- **蓝色批注**："副本入口 → 800m → 太远了"

### 小黑在干什么
小黑穿着全身限定外观在主城正中央打坐，聊天聊了 8 小时，副本一步没去。

## 12. 氪佬（Whale）

### 画面描述
小黑站在一个巨大的宝库/博物馆中央。四面墙上挂满了金光闪闪的限定武器、传说皮肤、绝版道具——每一件都完好无损地放在展示柜里。小黑正在用刷卡机（POS机）购买又一件新展品，表情认真得像是在做一件很严肃的事。宝库角落里有一个小小的、积满灰尘的"新手指引"卷轴（还没打开过）。小黑的角色等级牌挂在身上：Lv.1。宝库门外有一个"实际游戏内容"区域，但门上写着"改天"。

### 画面构成
- **主体（50%）**：小黑 + POS机 + 宝库展示墙
- **红色批注**："限定！买！"、"绝版！买！"
- **橙色批注**：画橙色箭头从 POS 机 → 展示柜，标注"氪金通道"
- **蓝色批注**："角色等级：1"、"拥有传说：47 件"

### 小黑在干什么
小黑正在刷卡购买又一件限定道具放进自己的博物馆，而角落里新手任务还没做。

## Prompt 使用说明

### 生图流程
1. 取对应类型的"画面描述"+"画面构成"+"小黑在干什么"
2. 拼接"Prompt 通用前缀"
3. 添加该类型的具体画面描述
4. 生成的 prompt 即为完整的单图生成指令

### 风格自查清单（生图后对照）
- [ ] 背景是纯白（#FFF），没有米色、纸纹、阴影、渐变
- [ ] 线条是黑色手绘感，不是完美矢量线条
- [ ] 小黑参与了核心动作——去掉小黑画面就不成立
- [ ] 留白充足，画面不拥挤
- [ ] 颜色只有黑+红+橙（蓝仅少量），没有多余颜色
- [ ] 没有 PPT 感、商业插画感、可爱卡通感
- [ ] 没有任何标题文字出现在左上角
- [ ] 不是流程图，不是架构图，不是信息图
