# 12 种玩家类型 — 完整形象 IP 提示词（单图生图即用）

> 每条 Prompt 均为完整独立的生图指令，可直接复制使用。
> 统一规格：16:9 横版，纯白背景，黑色手绘线稿，5-8 处手写批注。

---

## 类型 1：分奴（Score Slave）

**代号**：`score_slave`
**Bartle 象限**：杀手 · 竞技向
**一句话人设**：赢是氧气，输是窒息。Rank 分就是血压计。

### 完整生图 Prompt

```
16:9 横版中文正文配图。纯白背景（不是米色不是暖灰，没有任何纸纹、阴影、渐变）。
黑色手绘线稿风格，线条纤细略带抖动，不是机械矢量线条。
画面大量留白，主体占画面 45-55%。

画面内容：小黑（黑色实心块状小怪物，白色圆点眼睛，细黑线腿，柱形身体，
表情永远是空的、呆的、认真的）坐在一台巨大的"分数血压仪"前。
一根管子从仪器连到小黑的头上，仪表的指针在红色危险区疯狂抖动。
小黑双手紧握鼠标，身体紧张到微微弯曲（身体前倾到极限，但白点眼表情依然空白）。
屏幕上显示一个巨大的Rank标志和向下的箭头。
脚边散落着几个被捏碎的写着"娱乐"的球体。

小黑必须是画面的核心动作执行者，不是装饰。情绪通过身体姿态和周围符号表达，
面部永远不改。

5-8处中文手写批注：
- 红色："再输一把就睡"（旁边小字："第8次"）、"血压↑"、"T0 only"
- 橙色：从小黑到屏幕画一条橙色箭头，标注"胜负反馈"
- 蓝色："胜率焦虑"

不要商业插画感、不要PPT信息图、不要可爱卡通、不要复杂架构图、
不要标题文字在左上角。线条要怪诞、有趣、简洁清爽、天马行空。
```

**核心隐喻**：被分数绑架的人，娱乐已被碾碎。
**核心道具**：血压仪、鼠标、Rank 标志
**主色调**：红↑
